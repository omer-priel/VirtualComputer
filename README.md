VirtualComputer
Virtual Computer like CMD and is save the data on Virtual Hard Drive file (.vhd).

Directory Structure:
drives/drive0.vhd
drives/drive1.vhd
.vhd - Virtual Hard Drive

Commands:
exit - Exit the program
drives - Show drives list
dir - Show directory
md - Create directory
rd - Remove directory
cd - Change directory
mf - Create file
rf - Remove file
move - Move file or directory
copy - File or directory
rename - Rename file or directory
print - Show file
edit - Edit file
echo - Print text

* If not exist derive, create drive
* Can open number of virtual computers in the same time. But can't open on the same drives.
